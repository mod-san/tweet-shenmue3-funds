# Tweet Shenmue 3 Funds

This is hosted on Heroku.

The ENV refferences in the code, refer to environment variables.

**List of environment variables**

Database:
* DB_HOST
* DB_PORT
* DB_NAME
* DB_USER
* DB_PASSWORD
* DB_TABLE_NAME

Blitline:
* BLITLINE_APP_ID

Image:
* IMG_SRC

Twitter:
* CONSUMER_KEY
* CONSUMER_SECRET
* ACCESS_TOKEN
* ACCESS_TOKEN_SECRET

On Heroku, you can set those in the settings.
